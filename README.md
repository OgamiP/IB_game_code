# 自作3Dアクションゲーム「INF BATTLE」
Unity3Dで作成した自作アクションゲーム「INF BATTLE」
![manual](/image/main_image.png)

- 対応プラットフォーム:Windows PC
- 製作期間
    - プログラム:約1ヶ月 
    - 3DモデルやUI制作など:約1ヶ月
    - 以降少しずつ改良

***

##操作方法

[w]         前進

[s]         後退

[a]         右旋回

[d]         左旋回

[space]     ジャンプ（３段目まで。4段目以降で飛行モード遷移、以降spaceで高度上昇)

[e]         ローリング

[q]         急降下

[tab]       カメラ反転

[隠しキー1]  ガンマンモードに切り替え(β版仮実装中)

[隠しキー2]   ガンマンモード解除

[escape]  タイトルに戻る
